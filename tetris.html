First of all, you should probably already know how to play Tetris. If you don't, too bad.

You can use the arrow keys or WASD to move.

Hit the Save button after selecting a Difficulty option. The Difficulty will not update until you save. This is not necessary for changing Themes.

Cheat Mode is not a Difficulty or Theme option, I just ran out of space on this page to put it somewhere else. If you turn Cheat Mode on, you can move the tetrominoes even when the game is paused.

In order to leave the "rip" Difficulty option, you must select another Difficulty option and then hit Save when the Tetris game is visible. If you hit Save while the game is invisible or transparent, it will stay that way in the other Difficulty modes. The only way to reverse this is to go back to the "rip" mode and switch and hit Save when the game is fully visible. If you don't know what I'm talking about, go look at it.
